# Tatil Uygulaması Backend

Bu uygulama, tatil ve etkinlik yönetimi için geliştirilmiş bir backend sistemidir. Kullanıcıların tatil planlarını yönetmelerine olanak tanır ve etkinliklerin oluşturulması, düzenlenmesi ve yönetilmesi süreçlerini kolaylaştırır.

## Özellikler
- Kullanıcı hesapları oluşturma ve yönetme
- Etkinlik takvimi
- Tatil planlama aracı
- Bildirimler ve hatırlatıcılar
